# ibrahimsher18-AI-Fake-News-Detector


AI Fake News Detector

This project uses Machine Learning (Logistic Regression + TF-IDF)
to classify news as real or fake.

🔹 Features:
- Real-time prediction
- Confidence score
- Trained on Kaggle dataset

🔹 Tech Stack:
- Python
- Scikit-learn
- Streamlit

🔹 Accuracy:
~92%
